# Pasta de Pesquisa

Esta pasta contém os materiais de pesquisa individual desenvolvidos pelos
membros do grupo.

Cada subpasta corresponde a um membro do grupo e inclui a explicação dos
temas pesquisados, conceitos fundamentais e referências utilizadas como
apoio à implementação do caso prático.

## Observação
O código-fonte do projeto encontra-se na pasta `src/`.
